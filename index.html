YAML Format Guide

Define your infrastructure using a simple YAML structure:

name: My Infrastructure
description: Description of your infrastructure
components:
  - name: Component Name
    type: service/database/storage/network/compute
    properties:
      - key: value
    connections:
      - target: Other Component Name

Supported Types: service, database, storage, network, compute, gateway, cache, queue, or any custom type
